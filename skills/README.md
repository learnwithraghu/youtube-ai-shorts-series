# Course Skills Index

These are project-specific content skills for future work on the course.

## Available skills
- `section-architecture.md` — define and expand course sections
- `lesson-design.md` — turn sections into lessons and timeboxed flow
- `snippet-design.md` — create code snippet plans and implementation beats
- `slide-design.md` — convert content into slide-friendly structure
- `exercise-design.md` — create labs, prompts, and practice tasks
- `eval-dataset-design.md` — create realistic eval cases and rubrics

## Usage rule
Before creating downstream assets, start from the section guide and keep the shipping-assistant narrative consistent.
