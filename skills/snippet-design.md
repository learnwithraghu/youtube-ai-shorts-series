# Skill: Snippet Design

## Purpose
Create code snippet plans from the section and lesson structure.

## Output
For each snippet provide:
- goal
- context
- input/output
- key code blocks
- what learners should notice

## Rules
- Snippets must map directly to a lesson objective
- Show only the essential code needed to teach the concept
- Prefer readable, production-style examples
- Keep snippets aligned with the chosen stack
