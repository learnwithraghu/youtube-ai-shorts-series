# 02 — Progressive Coverage Strategy

## Purpose
Show how to expand the dataset deliberately over time.

## What this chapter teaches
- how to grow from core flows to broader coverage
- why order matters when adding examples
- how to avoid random dataset sprawl

## Topics covered
- normal cases first
- edge cases next
- risky cases last
- coverage planning by scenario type

## Talking points
- a dataset should grow like a product, not like a pile of examples
- each new case should add useful signal
- coverage strategy keeps evaluation focused

## Output
A structured growth plan for eval coverage.
