# 04 — Grounded Responses and Safe Fallback

## Purpose
Define the behavior policy the assistant should follow in every response.

## What this chapter teaches
- how to keep answers grounded in available data
- when to ask for more information
- when to escalate instead of guessing
- how to stay calm, clear, and helpful

## Topics covered
- grounded response style
- uncertainty handling
- safe fallback language
- escalation behavior
- tone and consistency

## Talking points
- confidence should come from evidence, not just wording
- safe failure is better than speculative answers
- good behavior needs to be defined before it can be evaluated

## Output
A response policy that later evals will enforce.
