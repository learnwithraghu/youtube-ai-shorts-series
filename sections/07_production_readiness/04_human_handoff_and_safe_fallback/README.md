# 04 — Human Handoff and Safe Fallback

## Purpose
Teach the assistant to fail safely when it cannot answer well.

## What this chapter teaches
- when to escalate to a human
- how to phrase fallback messages
- how to reduce risk when confidence is low

## Topics covered
- escalation paths
- fallback responses
- uncertainty handling
- risk reduction

## Talking points
- safe failure is better than a wrong answer
- handoff is part of a support system’s value
- the assistant should know its limits

## Output
A safer support workflow.
