# 03 — Stale Data and Drift

## Purpose
Explain how production systems lose quality over time.

## What this chapter teaches
- how stale data affects responses
- what model or behavior drift looks like
- why ongoing monitoring matters

## Topics covered
- stale shipment or policy data
- model drift
- behavior drift
- quality degradation over time

## Talking points
- production is not static
- even good systems decay without monitoring
- drift is a normal operational reality

## Output
A monitoring mindset for production.
