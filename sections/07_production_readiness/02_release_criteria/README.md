# 02 — Release Criteria

## Purpose
Define what good enough means for shipping the assistant.

## What this chapter teaches
- how to choose thresholds
- how to balance quality and risk
- how to make release decisions explicit

## Topics covered
- quality thresholds
- risk tradeoffs
- release checklists
- shipping decisions

## Talking points
- release criteria should be documented
- not every regression should ship
- clarity makes teams faster and safer

## Output
A practical release checklist.
