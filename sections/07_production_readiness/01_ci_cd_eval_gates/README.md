# 01 — CI/CD Eval Gates

## Purpose
Show how evals can block bad releases automatically.

## What this chapter teaches
- what a release gate is
- how evals fit into CI/CD
- why automated checks matter

## Topics covered
- automated evaluation checks
- build gating
- failure handling in pipelines

## Talking points
- shipping without gates is risky
- evals should influence release decisions directly
- CI makes quality enforcement repeatable

## Output
A release gate concept.
