# 02 — Score Summaries

## Purpose
Show how to summarize evaluation outcomes in a simple, readable way.

## What this chapter teaches
- how to aggregate results
- what metrics to include
- how to present pass/fail information

## Topics covered
- average scores
- pass/fail counts
- metric breakdowns
- concise reporting formats

## Talking points
- summaries should make the main story obvious
- different metrics should be visible together
- a clear summary reduces analysis friction

## Output
A concise summary view.
