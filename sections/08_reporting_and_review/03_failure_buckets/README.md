# 03 — Failure Buckets

## Purpose
Group similar failures so they are easier to review and prioritize.

## What this chapter teaches
- how to categorize failures
- how buckets help teams focus
- how to analyze failure patterns

## Topics covered
- categorization strategy
- prioritization by bucket
- recurring failure classes

## Talking points
- grouped failures are easier to fix than isolated examples
- buckets reveal systemic issues
- the goal is action, not just organization

## Output
A practical review structure.
