# 02 — Comparing Runs

## Purpose
Show how to compare evaluation results across different versions.

## What this chapter teaches
- how to read score deltas
- how to spot improvement or degradation
- how to compare runs consistently

## Topics covered
- run-to-run comparisons
- metric changes
- version diffs
- identifying meaningful shifts

## Talking points
- a higher score is not always better if another metric drops
- comparisons should focus on product impact
- run analysis helps make iteration disciplined

## Output
A repeatable comparison workflow.
