# 03 — Reference Answers vs Rubrics

## Purpose
Teach the two main scoring approaches we will use.

## What this chapter teaches
- when to use exact or reference-based checks
- when to use rubric-based judging
- how to choose the right scoring method for each failure type

## Topics covered
- reference answers
- rubric definitions
- structured judgment
- tradeoffs between strictness and flexibility

## Talking points
- not every good answer has one exact wording
- some failures need objective checks
- some behaviors need qualitative scoring

## Output
A scoring strategy matched to the product problem.
