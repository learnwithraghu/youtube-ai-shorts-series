# 04 — Core Metrics

## Purpose
Introduce the main quality dimensions that matter for the shipping assistant.

## What this chapter teaches
- which metrics map to product risk
- how to measure more than just correctness
- how to think about quality as a multi-dimensional system

## Topics covered
- correctness
- groundedness
- tone
- safety
- tool use

## Talking points
- a correct answer can still be unsafe or off-tone
- groundedness matters because the assistant must rely on real data
- tool-use quality is part of overall behavior quality

## Output
A practical metric set for the assistant.
