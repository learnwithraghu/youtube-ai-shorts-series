# 05 — Running and Reading Local Evals

## Purpose
Show how to execute evals and interpret what the results mean.

## What this chapter teaches
- how a local eval run works
- what score summaries tell us
- how to inspect failures and decide what to fix next

## Topics covered
- running evaluations locally
- reading aggregate scores
- inspecting individual failures
- turning results into debugging actions

## Talking points
- results should guide iteration, not just report status
- local evals are the fastest feedback loop
- failure analysis is part of product development

## Output
A working local evaluation workflow.
