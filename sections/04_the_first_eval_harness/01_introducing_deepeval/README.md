# 01 — Introducing DeepEval

## Purpose
Explain why DeepEval is the evaluation layer for the course and how it fits into the workflow.

## What this chapter teaches
- what an eval tool does
- why LLM evaluation is different from traditional testing
- how DeepEval helps structure quality checks

## Topics covered
- the role of an eval framework
- where DeepEval fits in the app lifecycle
- why scoring LLM behavior requires more than assertions

## Talking points
- we need measurable quality, not intuition
- eval tooling turns product expectations into repeatable checks
- DeepEval is how we connect behavior to metrics

## Output
A clear mental model of the eval stack.
